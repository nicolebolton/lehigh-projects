Web Font Classifications

Oldstyle Example
Resemble calligraphy (writing in ink).
Diagonal stress in the strokes.

Didone Example
Very high contrast between thick and thin strokes.
Vertical stress in the strokes.

Slab-serif Example
Heavy "brick-like" serifs.
Imperceptible differences between stroke weights.

Grotesque Example
Low contrast between thick and thin strokes.
Vertical or no observable stress.

Humanist Example
Resemble calligraphy (writing in ink).
Slanted stress.

Geometric Example
Low contrast between thick and thin strokes.
Vertical stress, and circular round forms.
